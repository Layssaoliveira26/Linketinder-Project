# Linketinder

O Linketinder é um sistema de conexão entre candidatos e empresas, desenvolvido em Groovy. O projeto simula uma plataforma de recrutamento onde é possível visualizar listas de candidatos e empresas cadastradas.

## Sobre o Projeto

O Linketinder é uma aplicação console que permite:
- Visualizar lista de candidatos com suas competências e informações profissionais
- Visualizar lista de empresas com descrições e dados corporativos
- Sistema de menu interativo para navegação entre funcionalidades

## Tecnologias

- **Groovy 5.0.0** - Linguagem de programação principal
- **Java** - Plataforma de execução

## Pré-requisitos

- Java JDK 8 ou superior
- Groovy 5.0.0
- IntelliJ IDEA (recomendado) ou outra IDE compatível


## Como Rodar a Aplicação

1. Execute a classe principal `Main.groovy`:
```bash
groovy src/groovy/Main.groovy
```

2. No menu, escolha uma das opções:
   - Digite `1` para visualizar a lista de candidatos
   - Digite `2` para visualizar a lista de empresas
